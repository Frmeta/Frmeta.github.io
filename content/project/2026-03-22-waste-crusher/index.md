Waste Crusher is a puzzle game where the players work at waste management site and they need to strategically fit waste into a landfill. The game features a tile-based grid where players arrange variety of waste types (normal, metal, toxic, and magnets).

{{< img src="waste-types.gif" >}}


Players are required to think outside the box by using machines (hydraulic press, cutter, and incenerator) to reshape the waste.

{{< img src="machine-types.gif" >}}
